# Archlinux `paru` Commands

## Installing new Packages

```bash
paru -S <package-name>
```

## Removing Packages

```bash
paru -Rs <package-name>
```

## Removing Packages (with Dependencies)

```bash
paru -Rs <package-name>
```

## Getting Package Info

```bash
paru -Si <package-name>
```
